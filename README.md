#项目名称：bibliometrics-project

关于文献计量学的项目仓库

##团队成员：郑城焌(组长)，尹佳乐，柏航，邱果林，余仁杰

##研究方向：可穿戴电子器件的热管理技术与智能调控系统研究

##项目计划：尹佳乐（负责检索与数据清洗）、柏航（负责计量分析与可视化）、邱果林（负责报告撰写）、余仁杰（负责开源项目二次开发）

项目计划
阶段一：需求分析与基础搭建

调研不同学科的文献计量学研究需求，明确核心功能优先级。
完成多源数据采集模块的原型开发，支持 CNKI 与 WOS 的数据导入。
搭建基础架构与开发规范，完成 GitHub 仓库初始化与文档编写。

阶段二：核心功能开发
实现数据标准化处理模块（格式统一、去重、缺失值填充）。
开发核心计量分析算法（关键词共现、合作网络分析、共被引分析）。
完成基础可视化模块（静态图表 + 简单交互式图谱）。
编写单元测试与示例案例。

阶段三：功能优化与开源测试
优化数据采集效率。
优化可视化交互体验，支持自定义布局与导出功能。

数据来源
公开学术数据库：CNKI（开放 API）、Web of Science（开发者 API）、OpenAlex（开源数据库）。
测试数据集：采用公开的文献计量学基准数据集
用户自定义数据：支持用户上传本地文献数据（CSV/Excel 格式）进行分析。

本项目为课程小组作业，旨在通过文献计量分析方法，掌握学术研究的基本流程，并实现研究结果的可复现性。
